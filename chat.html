 Project Workspace

            
The Project Workspace transforms Aetheria into a dedicated coding agent. It operates directly within your local file system, creating the ultimate pair-programmer experience without tedious copying and pasting.

            

            graph LR
                User([You]) -->|Command| AI[Aetheria Coder]
                AI -->|Modify| FS[(File System)]
                AI -->|Terminal output| Term[Terminal]
                Term -.->|Reads state| AI
            

            
 Superpowers: Native file creation, terminal execution, automatic repo cloning, and multi-file refactoring step-by-step.

        


        
        

            
 Computer Workspace

            
The Computer Workspace empowers Aetheria to operate your OS like a human. Utilizing screen vision and mouse/keyboard automation, the AI natively interacts with your apps and browsers.

            

            graph LR
                AI[Aetheria Agent] -->|Reads| Scr[Screen]
                AI -->|Clicks & Types| HID[Mouse/Keyboard]
                HID -->|Drives| App[Desktop Applications]
                App -.->|Updates UI| Scr
            

            
 Superpowers: Fully autonomous desktop workflows, visual web browsing, system configuration, and software testing.
