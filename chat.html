Cloud Sandbox Mode

                        
Isolated Linux container (`/home/sandboxuser`) via SandboxTools with full terminal access, file tools, and dedicated cloud compute.

                    

                

                

                    

                    

                    

                        
Local Desktop Mode

                        
Direct filesystem access via LocalCoderTools and an Electron bridge. Edits your files and runs commands natively on your machine.

                    

                

                

                    

                    

                    

                        
Native Git Control

                        
Read logs, check status, diff branches, and seamlessly version control your codebase directly from the active environment.

                    

                

                

                    

                    

                    

                        
Project Deployment

                        
Instantly deploy code from the cloud sandbox to live endpoints and query backend runtime databases using DeployedProjectTools.
